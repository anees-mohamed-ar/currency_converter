# Currency Converter Project

This project allows you to convert amounts between different currencies using real-time exchange rates.

## Features

- Get real-time exchange rates.
- Convert amounts between different currencies.
- User-friendly interface for easy currency conversion.

## Requirements

- Python 3.x
- `tkinter`
- `requests`

Install the required libraries:

```bash
pip install tk requests
```

## Files

- `main.py`: Entry point to run the currency converter application.
- `currency_converter.py`: Contains the code for the currency converter application.
- `reverse_currency_codes.py`: Contains the dictionary of currency codes and their corresponding names.
- `utils.py`: Utility functions (styles, etc.).

## How to Run

Run the main application:

```bash
python main.py
```

## How to Use

1. **Get Exchange Rates**:
   - Select the base currency.
   - Select the target currency.
   - Enter the amount to convert.
   - Click "Convert" to get the converted amount.
